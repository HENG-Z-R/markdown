
### 开始
| 中文       | 韩语   |
| --------   | -----:  | 
| 你好    | 안녕 하세요   | 
| 是不是本人？      |   본인이 맞나요?   |  
| 是的      |   그렇다   | 
| 我要开始录音了      |   나는 녹음을 시작하겠다  |  
| 好的  |  좋은  | 
| 请自我介绍      |  자기소개를 하시오 | 
| 非常感谢      |  감사합니다 | 
| 再见     |  안녕히 계세요 |
