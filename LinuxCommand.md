# Linux Command


** 基础命令 **


| 代码        |   功能  |  注释    |
| ----        |   ----  |    ----  |
| date       |   时间  |  系统端    |
| cal        |   日历  |      |
| cal xxxx        |  xxxx年日历  |      |
| cal n xxxx       |   xxxx年n月日历  |      |
| pwd        |   查看当前所在目录  |      |
| cd        |   返回总目录  |      |
| cd A        |   进入A目录  |      |
| cd ..        |   退出文件夹  |  后退一级    |
| ls |   查看目录下文件夹  |      |
| ls -l        |   查看文件夹详细内容  |      |
| mkdir A/      |   创建A文件夹  |      |
| rmdir A/      |   删除A文件夹  |      |
| cp A B       |   复制A文件夹  |  复制出的文件名为B    |
| cat A       |   查看A文件中内容  |      |
| diff A B       |   查异  |  仅显示差异部分    |
| head A -n x        |   查看前X行内容  |      |
| tail A -n x        |   查看后X行内容  |      |
| wc A       |   查文件大小  |   行数+单词书+字节（-w -l -c）   |
| clear        |   清屏  |      |
